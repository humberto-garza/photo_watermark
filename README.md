# Photo Watermark App

This project is a simple web application that allows users to upload photos and apply watermarks to them. The application is structured to provide a user-friendly interface for watermarking images stored in the "watermark" directory.

https://humberto-garza.github.io/photo_watermark/

## Project Structure

```
photo-watermark-app
├── index.html        # Main HTML document
├── css
│   └── styles.css    # Styles for the webpage
├── js
│   └── app.js        # JavaScript functionality for watermarking
└── README.md         # Project documentation
```

## Getting Started

1. **Clone the repository** or download the project files to your local machine.

2. **Open the project** in your preferred web browser by opening the `index.html` file.

3. **Upload Photos**: Use the interface to upload photos from the "watermark" directory.

4. **Apply Watermark**: Select the desired watermark and apply it to your images.

5. **View Results**: The watermarked images will be displayed on the webpage.

## Dependencies

This project does not have any external dependencies. It uses standard HTML, CSS, and JavaScript.
